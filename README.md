# Coleção de Livros

Aplicação de coleção de livros utilizada na Faculdade Impacta.

## Rodando a aplicação

### Backend

1. Acesse o folder backend

   ```PlainText
   cd backend
   ```

2. Instale as Dependências

   ```PlainText
   npm install
   ```

3. Inicie a aplicação

   ```PlainText
   npm run start:dev
   ```

4. A aplicação estará rodando em <http://localhost:3000/books>
